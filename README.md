# Blog Backend

Este proyecto es una práctica personal para desarrollar un blog desde cero, con el objetivo de mejorar mis habilidades en desarrollo backend y frontend. Estoy utilizando **TypeScript** para programar toda la lógica del proyecto, mientras aprendo e implemento herramientas modernas para el diseño y estilización.

El blog se centrará en compartir publicaciones relacionadas con mi aprendizaje personal y avances en programación, documentando tanto el proceso técnico como las tecnologías exploradas.

## Objetivos del proyecto

1. **Practicar el desarrollo backend:** Crear una API robusta para manejar publicaciones, comentarios y suscripciones, completamente desarrollada en **TypeScript**.
2. **Aprender sobre diseño y estilización:** Utilizar **Sass** para gestionar los estilos de manera modular y escalable, y explorar frameworks CSS como **Bootstrap** o **TailwindCSS** para definir el diseño del frontend.
3. **Explorar buenas prácticas:** Aplicar principios de seguridad, modularidad y escalabilidad en el desarrollo de este proyecto.

## Tecnologías utilizadas

- **TypeScript (TSC):** Para escribir código tipado y estructurado en todo el proyecto.
- **Sass:** Para estilización avanzada y gestión eficiente de CSS.
- **Framework CSS (por definir):** Planeo integrar **Bootstrap** o **TailwindCSS** como apoyo al diseño, además de **Sass**.
- **Node.js** y **Express:** Para la implementación del backend.
- **Bases de datos relacionales:** Para el almacenamiento de datos del blog.

## Estado del proyecto

El desarrollo está en curso. Actualmente estoy en la fase de planificación e integración de tecnologías clave, explorando opciones para el framework CSS y definiendo la estructura del proyecto.
